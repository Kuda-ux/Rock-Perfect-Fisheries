# Rock Perfect Fisheries ZW — Website

Premium, conversion-focused website for **Rock Perfect Fisheries ZW**, Zimbabwe's aquaculture partner for pond construction, dam liners, fingerlings, fish feed, water chemicals, predator nets and free on-site training.

## Tech Stack
- **React 18** + **Vite 5**
- **Tailwind CSS 3**
- **Lucide React** icons
- Inter + Plus Jakarta Sans (Google Fonts)

## Run Locally
```bash
npm install
npm run dev
```
Visit http://localhost:5173

## Build
```bash
npm run build
npm run preview
```
